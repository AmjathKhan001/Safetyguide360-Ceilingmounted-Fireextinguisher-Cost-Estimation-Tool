# Modular Fire Extinguisher Calculator

A professional web tool for calculating ceiling-mounted modular fire extinguisher requirements and generating commercial quotes.

## Features

- **User-Friendly Interface**: Clean, modern design with intuitive controls
- **Real-time Calculations**: Instant calculation of extinguisher requirements
- **Multiple Export Options**: Generate PDF and Excel quotes
- **Room Management**: Add, edit, and remove rooms with area calculations
- **Professional Quotes**: GST-inclusive pricing with detailed breakdowns
- **Responsive Design**: Works on all devices

## How to Use

1. **Enter Organization Details**: Fill in vendor and client information
2. **Configure Product**: Select extinguisher agent and cylinder size
3. **Add Room Dimensions**: Input room sizes to calculate coverage
4. **Generate Quote**: Click calculate to view results and export options

## Deployment

This project is ready for deployment on Vercel:

1. Push this code to a GitHub repository
2. Go to [Vercel](https://vercel.com)
3. Import your GitHub repository
4. Deploy with default settings

## Technologies Used

- HTML5, CSS3, JavaScript (ES6+)
- Tailwind CSS for styling
- jsPDF for PDF generation
- SheetJS for Excel export
- Font Awesome icons
- Google Analytics integration
